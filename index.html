Jurassic Outfitting Co.

This was the first proper website I ever created, a culmination of all the HTML and CSS skills I had learned throughout the semester. It was developed to be a part of the official Jurassic Park website if it ever opened in real life, providing gear and equipment to all those that visited. This project was unlike any others I have done in school, being by far the largest scale project I had ever done (2 months). This quickly led to a struggle to stay on track, but my partner and I quickly learned how to effectively collaborate by splitting up work according to our strengths and weaknesses. Following this project, I became well accustomed to structuring and styling websites.

EECU Budget Calculator

After dipping my toes in Javascript for a few weeks, I got an opportunity to create a full scale site with HTML, CSS, and Javascript when the Educational Employees Credit Union (EECU) requested us to design and develop an online budget calculator. Following this project, I gained a solid understanding of manipulating the DOM and how to use external libraries because I used chart.js. For the early stages of this project, students were required to develop the sites of other students based on their Figma designs and their individual instructions. This experience helped me to learn how to translate the ideas of other people into code for a website

Battleship Unknown

Battleship Unknown is a web based adaptation of the board game battleship with its own unique twists. This project had me use all of the web development skills that I had gained throughout the year, including market research, wireframing in Figma, creating flowcharts, and then programming with HTML, CSS, and Javascript. This project really fine-tuned my ability to manipulate HTML and CSS with Javascript through my use of CSS animations and managing game states. I was primarily responsible for programming the games logic using javascript, while my partner focused more on styling the website.
